  • 渲染树绘制（paint）

 
重排(reflow || relayout)  
  • 定义：部分/整个渲染树需要重新分析并且节点尺寸需要重新计算

重绘(repaint)

  • 定义：节点的【几何属性】发生改变或样式发生改变

合成层(composite)

  • 定义：对页面中 DOM 元素的绘制是在多个层上进行的。在每个层上完成绘制过程之后，浏览器会将所有层按照合理的顺序合并成一个图层，然后显示在屏幕上。

bstyle.backgroundColor = "#fad"; // 重绘
bstyle.fontSize = "2em"; // 重排+重绘

  • 访问样式信息会组织浏览器优化重排并立即刷新队列
    • offsetTop, offsetLeft, offsetWidth, offsetHeight
    • scrollTop/Left/Width/Height
    • clientTop/Left/Width/Height
    • getComputedStyle(), or currentStyle in IE
    • 以上就是一个节点基本的可请求信息。无论何时发出请求，浏览器需要计算最新的值。为了实现这一目的，浏览器需要应用所有队列中的变更，刷新队列然后去实现重排。
